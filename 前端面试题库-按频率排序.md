### 按公司特点
- **字节跳动**：偏重算法题 + 代码输出题 + 网络基础，面试轮次多（通常3面+HR面）
- **腾讯**：网络安全（XSS/CSRF必考）+ 性能优化 + Vue/React 深入原理
- **美团**：CSS 布局 + Vue 全家桶 + 事件循环输出题
- **百度**：CSS 基础考得细（盒模型/BFC/布局）+ 手写代码
- **拼多多**：算法题偏多 + ES6 基础 + 闭包/原型链

### 备考优先级
1. **JS 基础是重中之重**：闭包、原型链、this、Event Loop、类型判断占了面试题的 36%
2. **CSS 布局必须熟练**：居中、两/三栏布局、flex、BFC、盒模型是高频考点
3. **Vue 3 全家桶要深入**：Composition API、Pinia、Vue-Router 4、响应式原理（Proxy）
4. **TypeScript 现在几乎必考**：type vs interface、泛型、Utility Types
5. **网络知识不能忽略**：HTTP/HTTPS、TCP/UDP、跨域、缓存、状态码
6. **手写代码能力**：防抖节流、深拷贝、Promise、数组去重、call/apply/bind、快排
7. **算法准备**：合并有序数组/链表、反转链表、二叉树遍历、两数之和、斐波那契
8. **工程化**：Vite 原理、monorepo、ESLint 配置、CI/CD 流程
